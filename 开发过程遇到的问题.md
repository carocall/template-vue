# 1对于侧边栏的logo底部描边和header底部描边没有对齐的问题
-- 对于css样式,如果添加了
```css
.c{
    border-bottom: 1px solid #e5e7eb;
}
```
默认是沿着边缘向外拓展,需要添加
```css
.c{
    border-bottom: 1px solid #e5e7eb;
    box-sizing: border-box
}
```
才能让加上描边的大小和原本的div一致

# 关于滚动条

## 隐藏滚动条
```css
.hide-scroll-bar{
    /* 隐藏滚动条但保持滚动功能 */
    overflow-y: auto;
    scrollbar-width: none; /* Firefox */
    &::-webkit-scrollbar {
        display: none; /* Chrome/Safari/Edge */
    }
}
```

# 关于侧边栏的logo和menu同时滚动,而不是logo固定,剩余区域可以滚动
因为一开始的结构是
```html
<el-aside>
    <el-menu>
        <div>
            这里放logo区域
        </div>
        <el-menu-item>
            这里放menu-item
        </el-menu-item>
    </el-menu>
</el-aside>
```
所以不能滚动,需要改成
```html
<el-aside class="aside">
    <div class="logo">
        这里放logo区域
    </div>
    <el-menu class="menu">
        <el-menu-item>
            这里放menu-item
        </el-menu-item>
    </el-menu>
</el-aside>
```
才能滚动,此外还有一点,css样式需要确保logo高度限定,menu恰好占满全部,可以用calc函数
```css
.aside{
    height: 100%;
    width: 200px;
}
.logo{
    height: 60px;
}
.menu{
    height: calc(100% - 60px);
    /*并且要允许滚动*/
    /* 隐藏滚动条但保持滚动功能 */
    overflow-y: auto;
    scrollbar-width: none; /* Firefox */
    &::-webkit-scrollbar {
        display: none; /* Chrome/Safari/Edge */
    }
}
```
